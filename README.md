# Bevin and Jaimi

A beautiful wedding invitation web app for Bevin Binny and Jaimi Benny. It presents the event details, countdown, and RSVP flow in a polished, mobile-friendly experience with a floral themed design and celebration feedback after submission.

## Getting Started

Install dependencies and start the local dev server:

```bash
bun install
bun run dev
```

## Scripts

- `bun run dev` - start the development server
- `bun run build` - create a production build
- `bun run preview` - preview the production build locally
- `bun run lint` - run ESLint
